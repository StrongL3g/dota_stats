## 🔍 О проекте

Стандартные API Dota 2 и OpenDota не предоставляют доступ к истории матчей, если игрок скрыл свою статистику в настройках приватности. **Dota 2 Stats Harvester** обходит это ограничение, используя гибридный подход:

1. **JavaScript-скрипт** выполняется в браузере на странице Steam GCPD и извлекает идентификаторы матчей напрямую из локального хранилища.
2. **Python-оркестратор** использует библиотеку `dota2` для подключения к координатору игры и получения детальной статистики: герои, K/D/A, результат матча и другие параметры.

> 💡 Идеально подходит для личного анализа прогресса, подготовки отчётов или интеграции с другими инструментами аналитики.

## ✨ Возможности

- ✅ Сбор статистики для аккаунтов со **скрытой историей матчей**
- ✅ Экспорт данных в удобный CSV-формат
- ✅ Поддержка маппинга героев (человекочитаемые имена)
- ✅ Минимальные зависимости и простая настройка
- ✅ Разделение процесса на этапы для гибкости использования

---

## ⚙️ Требования

- Python 3.8 или выше
- Браузер с доступом к Steam (Chrome, Firefox, Edge)
- Аккаунт Steam с включённой двухфакторной аутентификацией *(рекомендуется)*
- Доступ к странице: `https://steamcommunity.com/my/gcpd/570`
