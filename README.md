# 🎮 Dota 2 Stats Harvester

Инструмент для автоматизированного сбора и анализа детальной статистики матчей Dota 2 из личной истории игрока (GCPD).

---

## 📌 Описание проекта
Проект решает проблему ограниченного доступа к истории матчей через стандартные API для аккаунтов со скрытой статистикой. 
Используется гибридный метод:
1. **JS-скрипт** — извлекает ID матчей напрямую из вашего браузера (Steam GCPD).
2. **Python-оркестратор** — подключается к координатору Dota 2 и выкачивает детальную статистику (K/D/A, герои).

## 🚀 Быстрый старт

### 1. Установка зависимостей
```bash
pip install dota2 steam python-dotenv
````

### 2\. Настройка конфига

Создайте файл `.env` в корневой папке:

```env
STEAM_USERNAME=ваш_логин
STEAM_PASSWORD=ваш_пароль

PATH_INPUT_MATCHES=dota_ids_for_api.csv
PATH_OUTPUT_REPORT=analytics_report.csv
PATH_HERO_MAP=hero_id_map.json
```

### 3\. Сбор данных

1.  Откройте в браузере [страницу истории матчей Steam](https://www.google.com/search?q=https://steamcommunity.com/my/gcpd/570/%3Fcategory%3DMatches%26tab%3DPlayerMatchHistory).
2.  Нажмите `F12`, перейдите в **Console** и вставьте JS-скрипт (находится в папке scripts или в описании ниже).
3.  Дождитесь окончания сбора, введите `stop()` и сохраните файл как `dota_ids_for_api.csv`.
4.  Запустите основной скрипт:

<!-- end list -->

```bash
python run_analytics.py
```

## 🛠 Структура проекта

  * `run_analytics.py` — "мозг" проекта, оркестратор всех этапов.
  * `hero_id_map.json` — справочник имен героев.
  * `analytics_report.csv` — итоговая таблица с вашей статистикой.
